## 🚀 自分のCSVで学習する手順

1. **CSVを用意する**
   - 1列目に「問題」、2列目に「答え」を書いたCSVファイルを作成する。
   - iPhoneの「ファイル」アプリに保存します。
   - 作成に迷った際は、[サンプルファイル](https://github.com/milgarden/flash_for_HJU/blob/main/kodomo_sample.csv)を参考にしてください。
2. **アプリを開く**
   - [アプリを起動する](https://milgarden.github.io/flash_for_HJU/)  <br>
　　　
   - うまく動かない方は　<br>
      https://milgarden.github.io/flash_for_HJU/  <br>
     をSafariにコピペして開いてください。<br>

   - Safariの共有メニューから「ホーム画面に追加」を推奨します。

3. **ファイルを読み込む**
   - 「ファイルをえらぶ」から作成したCSVを選択し、「学習をはじめる」をタップ！

## 🛠 FAQ - よくある質問と解決策
Q1. 画面が真っ白、またはボタンが反応しません。  
A. ブラウザのキャッシュが影響している可能性があります。ページをリロード（再読み込み）するか、一度ブラウザを完全に閉じてから再度URLを開いてみてください。また、JavaScriptが無効になっていると動作しません。

Q2. マイクが声を拾ってくれません（音声認識が動かない）。  
A. 以下の点を確認してください：  
-マイクの使用許可: ブラウザから「マイクの使用」を求められた際、必ず「許可」を押してください。  
-対応ブラウザ: iPhoneの場合は Safari、PC/Androidの場合は Google Chrome を推奨します。  
-静かな環境: 周囲の雑音が大きいと、声が正しく認識されないことがあります。  
-SSL (https): セキュリティ上の理由から、http:// ではなく https:// で始まるURLでアクセスする必要があります。  

Q3. CSVファイルを読み込んでも「準備完了」になりません。  <br>
A. CSVのフォーマットが正しいか確認してください。  
-1列目に「問題（解説）」、2列目に「答え（単語）」が入っている必要があります。  
-文字コードが UTF-8 で保存されていることを確認してください（Excelで保存する際、「CSV UTF-8 (コンマ区切り)」を選択してください）。

Q4. 外部URL（?csv=...）から読み込めません。  
A. 読み込み先のサーバー（GitHubなど）が外部からのアクセスを許可していない（CORSエラー）場合があります。  
-GitHub上のファイルを使う場合は、raw.githubusercontent.com から始まる「Raw」のURLを使用しているか確認してください。

Q5. 音声が途中で止まってしまいます。  
A. ネットワーク環境が不安定な場合や、ブラウザの省電力モードが有効な場合に音声合成が途切れることがあります。一度画面をタップして手動で進めると、再度動き出すことが多いです。
